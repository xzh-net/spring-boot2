# 代码炼金术：让程序自己编写程序

JavaParser 是一个开源项目,旨在为Java 1.0到Java 21提供全面的解析和分析功能。它不仅可以解析Java源代码,生成抽象语法树,还提供了丰富的API来操作和分析AST

核心功能：
- 代码解析
  - JavaParser可以将Java源代码解析成抽象语法树(AST)。这个AST包含了源代码的所有结构信息,使得后续的分析和操作变得非常方便。

- 符号解析
- AST操作
- JSON序列化

应用场景：
- 代码静态分析
- 自动生成代码
- 文档生成工具
- 依赖关系分析
- IDE智能提示插件
